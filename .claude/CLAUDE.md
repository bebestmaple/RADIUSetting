# RADIUS 认证客户端 - 项目指引

## 项目概述

跨平台 RADIUS/802.1X 网络认证客户端，支持一键启用认证、自动配置和诊断工具。

**目标平台**：
- Windows：7/XP/10/11（GUI）
- macOS（GUI）
- Linux：Ubuntu/Debian/CentOS7（GUI + CLI）

**核心功能**：
1. 一键启用/禁用 RADIUS 认证
2. 自动配置网络连接（802.1X）
3. 认证状态监控
4. 网络诊断工具

## 开发规范

### 代码质量
- 跨平台代码必须在目标系统上实测验证
- 网络配置操作必须支持回滚机制
- 敏感信息（密码、证书）必须加密存储
- 所有网络请求必须设置超时

### 测试要求
- 单元测试覆盖核心逻辑
- 集成测试覆盖网络配置流程
- 在至少 3 个目标平台上进行端到端测试

### 安全规范
- **禁止明文存储凭据**
- 使用系统 Keychain/Credential Manager 存储密码
- 证书验证必须严格（防中间人攻击）
- 管理员权限最小化（仅必要操作时提权）
